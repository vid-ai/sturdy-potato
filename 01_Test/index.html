The Basic Language of the Web : HTML

All the modern websites and web applications using three fundamental technologies: HTML, CSS and Javascript. These are three languages of the web.

In this post, we will focus on HTML. We will learn all about HTML and why you to should learn it.

What is HTML?

HTML stands for HyperText Markup Language. It is a markup language the web devlopers to structure and describe the content of the webpage( not a programing language

HTML consists of elements that describe the different types of content: paragraph, links, images, headings, videos, etc. Web browsers understand HTML and render HTML code as websites.

In HTML each elements are made up of three parts:

  • The opening tag
  • The closing tag
  • The actual element

    • Why should you learn HTML ?

    There are countless reasons for learning the fundamental language of the web. Here are 5 of them :

    • To be able to use the fundamental web dev language
    • To hand-craft beautiful websites instead of relying on the tools like worpress or wix
    • To build web applications
    • To impress friends
    • To have fun
